# upc_auto

[ ENGLISH ]

Script for automatic search of wifi networks belonging to UPC based on EVW3226 routers.

The following are required for correct operation:

aircrack-ng package

database 0xDEADCODE: (https://ubee.deadcode.me/db/keys.db.bz2)
